# Cripto_Moedas_Cotação

# 💹 Crypto Quotes App

Um aplicativo desenvolvido em **Flutter** para acompanhar cotações de criptomoedas em tempo real.  
O objetivo do projeto é oferecer uma interface simples, responsiva e intuitiva para que usuários possam visualizar o preço das principais moedas digitais.

---

## 🚀 Visão Geral

Com o **Crypto Quotes App**, você pode acompanhar rapidamente o valor de diferentes criptomoedas de forma atualizada.  
O projeto foi criado para estudos e prática de Flutter, mas já possui potencial para evolução com recursos avançados, como alertas personalizados e integração com carteiras digitais.

---

## ✨ Funcionalidades

- 📊 Listagem das principais criptomoedas (ex: Bitcoin, Ethereum, etc.)
- 📱 Layout responsivo e adaptado para diferentes tamanhos de tela

---

## 🛠️ Tecnologias Utilizadas

- [Flutter](https://flutter.dev/) - Framework principal
- [Dart](https://dart.dev/) - Linguagem de programação
- [HTTP](https://pub.dev/packages/http) - Requisições à API
- [Provider](https://pub.dev/packages/provider) - Gerenciamento de estado
- [API de Cotações](https://www.coingecko.com/en/api) - Fonte dos dados de preços
